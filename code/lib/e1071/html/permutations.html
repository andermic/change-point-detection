permutations {e1071}R Documentation

All Permutations of Integers 1:n

Description

Returns a matrix containing all permutations of the integers 1:n (one permutation per row).

Usage

permutations(n)

Arguments

n

Number of element to permute.

Author(s)

Friedrich Leisch

Examples
